# Course Overview

**Data Management 1** is an introductory course that builds practical skills for handling data responsibly. Students learn how to explain data management challenges, interpret **FAIR principles**, and understand how **metadata** improves discovery and reuse.

## Learning approach

The course uses small, reproducible examples and short guided exercises. The emphasis is on:
- conceptual clarity (with plain language definitions),
- practical inspection of dataset documentation, and
- lightweight processing of metadata examples in Python (offline).

## What is in this repository?

This repository is organized as a **4-week teaching plan**:
- **Week 1** is fully implemented with lessons, handouts, exercises, notebooks, and runnable example files.
- **Weeks 2–4** are scaffolded as placeholders so you can extend the course later.

See `teaching_plan_4_weeks.md` for the overall weekly structure.

## How instructors can use the materials

- Assign the relevant handouts for each day.
- Use the instructor notes to time discussion and address misunderstandings.
- Use the notebooks as consistent “reference implementations” for Python parsing tasks.
