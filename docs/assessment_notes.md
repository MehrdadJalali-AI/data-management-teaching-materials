# Assessment Notes (Week 1)

This repository includes instructor-facing guidance for Week 1 exercises and assessment.

## What is assessed

- **Day 1 practical (FAIR assessment):** students identify a persistent identifier (e.g., DOI), then provide a short FAIR-oriented result summary with strengths and improvement suggestions.
- **Day 2 exercises (metadata):**
  - Part A (in class): students interpret provided metadata examples and complete an improved Dublin Core record.
  - Part B (homework): students create lightweight DataCite XML and schema.org JSON-LD metadata files for a dataset of their choice.
