# Week 1 (Fully Implemented)

Week 1 contains two sessions:
- **Day 1:** Foundations of data management and the FAIR principles (Findable, Accessible, Interoperable, Reusable), ending with a hands-on FAIR assessment practical.
- **Day 2:** Metadata concepts and practical understanding, including multiple notebooks that parse example metadata standards locally, plus one structured in-class exercise and one homework exercise.
