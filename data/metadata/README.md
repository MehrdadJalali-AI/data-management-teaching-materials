# Example Metadata Files (Teaching-Only)

This folder contains small metadata examples for a single teaching dataset:

**“2025 Global Climate Data”**

All three standards below describe the same conceptual dataset so students can compare how the field names and structures differ.

## Files

- `climate_dataset_dublin_core.xml`
- `climate_dataset_datacite.xml`
- `climate_dataset_schemaorg.jsonld`

Additionally, short “generic example” metadata files are provided for quick reference:
- `dublin_core_example.xml`
- `datacite_example.xml`
- `schemaorg_example.jsonld`

## Offline-first design

These files are intended to be parsed locally with Python using the helper scripts in `src/`.
