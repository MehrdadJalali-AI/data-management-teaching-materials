# Discussion Questions (Week 1)

1. What is the difference between storing data and managing data?
2. Which data management challenge have you seen most in practice, and why?
3. Does a dataset need a DOI to be Findable? What alternatives exist?
4. If a dataset is open-access but poorly documented, which FAIR principle is most affected?
5. If metadata is available but data access is restricted, does the dataset still count as “Accessible”?
6. What does “Interoperable” require besides using a common file format?
7. Give one example of metadata that improves Reusable.
8. Why can two datasets share similar fields but still be hard to combine?
9. How might administrative metadata reduce risk for data users?
10. Which metadata standard would you choose first if the goal is DOI-based citation?
